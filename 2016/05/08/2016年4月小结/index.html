2016年4月小结

发表于 2016-05-08   |   分类于 总结   |     |   阅读次数

作者：Dimon

  • 微博：@Dimon-喰
  • LOFTER：@Dimon、

小结

  • Gan物妹已经发布，成功获得55个Star,15个Fork,是一个很好的开端。
  • 上个月在广州，这个月在江门待了整个月，都在充实自己的Android知识，虽然时有去浪，作息非常不规律，但是还是成功地迈开了第一步，加油！！！

习惯

根据上个月的规划，开始对网站文章减少涉猎，只对于有用的文章进行阅读，加快了工作效率。

这个月继续如此，并且来到了大深圳开始找工作（好吧，第一个晚上就睡不着失眠了…）

因为机缘巧合，了解到德鲁克大师的一些思想，逐步开始对自己的时间进行规划与跟踪记录。现在每天跟随我的不是手机，而是一本绿色的小本本，里面记录的是自己一天中时间到底去哪了，复阅当天的时间记录对比规划，然后不断修订对第二天的规划，将时间用到刀刃上（比如现在失眠，就跑起来码字，呵呵。）

阅读

  • Android群英传：49/342 17号开始陆续看的进阶书籍，群英传中自定义View等知识非常好，正在缓慢提升。
  • Android开发艺术探索：72/507 也是17开始的，这本相对偏向理论性，也是帮助极大，加深了自己对Android更深层的一些细节的了解。IPC机制方面的知识可以说是看过的最舒服的讲解了，~(≧▽≦)/~赞！

4月OKR反馈

  • 算法问题没有完成
  • 项目制作完毕
  • 到现在为止连续commit32天，除了清明两天没有敲代码外，每天都有相应的代码输出，虽然有时候实在是没什么东西commit，就会出现一些update README 这些…但是不管怎么样，我还是坚持了。
  • 作息问题基本随缘…经常有灵感舍不得睡觉…

5月OKR

Objective：

  • 对自己的时间消费进行实时记录
  • commit挑战继续，60天进发
  • 拿到offer

Key：

  • 准备两张单子，紧急事件的单子和必须要做但是却不是特别重要的事情单子
  • 每日早起第一件事就是检查每日粗略规划

Results：

  • 自身时间观念的提升
  • 砍去无谓的时间浪费
